TURBO CYBERSECURITY BOT

Turbo Security is a beginner-friendly console-based chatbot designed to help users learn and explore fundamental cyber security concepts. The chatbot responds to common questions about online safety, malware, phishing, password security, and general cyber security best practices.

Features

-Bot that helps users with cybersecurity relating to their queries like phising and more
-Interactive Chat for users to use for their curiousity
-wav file that makes it more enganing for users
-simple and easy to follow commands/prompts for the user to navigate easily through interactions with the bot
